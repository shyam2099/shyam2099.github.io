You only live once

2D side-scrolling game where you have to collect a certain number of items and avoid enemies to get access to a door and reach the next level.

The name was given because of the single life that you get to complete all the levels. Each level gets significantly more difficult than the previous one.

Bomberman

Recreate the classic game bomberman in Unreal Engine. 2 player shared screen game where the objective is to defeat the other player using the bombs as weapons.

Different abilities and powerups were added to change the range and effectiveness of the bombs and the player.

CORE

CORE was a cancelled third person MOBA inspired heavily by Epic Games' cancelled MOBA Paragon. After Paragon was shut down in April 2018, Epic Games released the assets and I was part of a team that was recreating Paragon using these assets.

On this team, I was responsible for creating the Jungle Creep AI. This includes spawn patterns and times, attack ranges, attack behaviour, and idle behaviour.

It was made with Unreal Engine, the AI was programmed using Unreal's Behaviour Trees and Environment Query System.

Claydo

Lead programmer. Volumetric sculpting app using primitive shapes built in Unity.

Uses SDF compute shaders to create the polygons and then combined inside a volume that allows a user to create models using boolean operations.

Other features worked on include:

- Camera controls with intuitive and smooth movement.

- Export models as .stl or .obj formats.

- Custom file type to save and load models within the app.

- MacOS port.

Anima Toon

Voxel style Animation App built in Unity.

Lead programmer for this app. Uses voxel volumes with a rigged skeleton and allows a user to model and animate the rig without having to leave the app.

Includes keyframing, saving poses, interpolation between keyframes, and onion skinning.

A modelling mode allows users to update the model at any time without affecting animations.

Other features worked on include:

- Custom made rigs for various skeletons including humanoid, and quadrupedal.

- Exporting full animation via gltf format.

- Exporting current viewing animation as a video file.

- Save and share the animation in a custom built format for AnimaToon.

Easy Sketch Pose

Combined Drawing and 3D posing application built in Unity.

Worked on the 3D posing half of this app. Built in humanoid model can be posed using controllers. Other obj files can be imported to add extra content. Exports back into the app in the 2D view as a new layer.

Other features worked on include:

- Physical camera controls and safe area view to show exactly what will be taken to the 2D view as a layer.

- Multiple shading modes like toon shaded, silhouette, outline only etc.

- Automatic model manipulation controls added for imported objects during runtime.
